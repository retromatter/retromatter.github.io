# junl.ee

Personal site of Junlee — full stack developer. Built with Jekyll and [jekyll-theme-console-bulma](https://github.com/retromatter/jekyll-theme-console-bulma).

## Stack

- **Jekyll** ~> 4.4
- **Theme** — jekyll-theme-console-bulma (terminal-inspired, Bulma CSS)
- **Plugins** — jekyll-feed, jekyll-sitemap, jekyll-seo-tag, jekyll-postfiles, jekyll-paginate
- **Hosted** — GitHub Pages

## Running Locally

```bash
git clone https://github.com/retromatter/retromatter.github.io.git
cd retromatter.github.io
bundle install
bundle exec jekyll serve
```

Navigate to `http://localhost:4000`.
